Aviso do Windows

Esta versão beta ainda não possui certificado comercial de assinatura de código. Por isso, o SmartScreen pode mostrar Fornecedor desconhecido. Confira se o arquivo foi baixado desta página e valide o SHA-256 antes de escolher Mais informações > Executar assim mesmo.
