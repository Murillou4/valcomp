Valcomp beta

Sua loja diaria do VALORANT no celular.

Instale o APK Android, abra o companion no Windows uma vez e vincule sua conta Riot com um codigo curto. Depois disso, o app consulta o backend hospedado e envia alertas quando uma skin desejada aparecer.

Baixar APK Android Baixar Companion Windows
1

Instale o app Android

Baixe o APK, permita a instalacao de apps externos no Android e abra o Valcomp. Crie uma conta ou entre com email e senha. Por enquanto nao usamos Google, GitHub ou Apple.

2

Gere o codigo

No app, entre em Vincular e toque em gerar codigo. Ele expira rapido de proposito, entao deixe o PC com Riot Client logado por perto.

3

Abra o companion

No Windows, extraia o ZIP inteiro e execute Valcomp Companion.exe. Cole o codigo do app. O companion detecta sua sessao Riot local e completa o vinculo com o backend.

4

Ative alertas

No app, entre em Alertas, registre notificacoes do aparelho e escolha as skins desejadas. Quando aparecerem na loja diaria, o backend envia uma notificacao.

Backend

API em producao: https://valcomp-api-cda2.fly.dev. O backend usa Supabase Auth/Postgres e endpoints internos do cliente Riot, sem Riot Developer API key.

Se o app mostrar relink_required, abra o companion de novo no PC logado na Riot e gere um novo vinculo.
